To start the application:
1) start the DB: java -cp h2-1.4.190.jar org.h2.tools.Server
2) make sure that DB path is right - jdbc:h2:tcp://localhost/~/IdeaProjects/employee/data
credentials : root 
3) run maven build 
4) open app at localhost 8080
